> **AgriSmart** is a modern, sleek P2P agricultural marketplace built with MongoDB, Express, React (Vite), and Node.js. It features a transparent bidding system, location-based farming, direct secure communications, and a built-in AI Chatbot assistant to support fair trade!

---

### ✨ Features
- 🚜 **Producer & Consumer Dashboards**: Specially crafted experiences for buyers and sellers.
- ⚡ **Real-Time Live Bidding**: WebSockets powering secure, instant auction bids.
- 💬 **Live Chat & AI Bot**: Direct P2P negotiation alongside a helpful 24/7 AI agriculture assistant.
- 📍 **Map-Based Marketplace**: Built-in Leaflet Maps for visualizing distance to fresh crops.
- 🔒 **Secure Authentication**: JWT-secured login portal with sleek UI mapping.

### 🛠 Tech Stack
- **Frontend**: React.js, Vite, Tailwind CSS, Framer Motion, Leaflet
- **Backend**: Node.js, Express.js, MongoDB (Atlas), Socket.io
- **AI Integration**: Groq (LLaMA 3)

### 🚀 Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shruti-ghoshhhh/AgriSmart.git
   cd AgriSmart
   ```

2. **Run Backend:**
   ```bash
   cd backend
   npm install
   npm start
   ```

3. **Run Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
